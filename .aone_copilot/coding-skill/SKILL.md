# coding-skill

## 指令说明
* 使用方法：当识别到用户意图为编码实现时，会自动触发此技能并加载 REFERENCE 文件夹中的一个或几个相关的参考文件作为规范约束，以帮助用户编写高质量的代码。
* 参考文件：在文件夹`.aone_copilot/skills/coding-skill/REFERENCE`下，采用分层设计的规范文件，具体说明如下：
  - `编码实现spec.md`：用于代码实现的约束规范，以产出高质量、易维护、易扩展的代码

#### 默认编码流程
- 读取 `PROJECT_CONTEXT.md`：获取工程结构设计，使得代码设计符合工程架构设计
- 读取相关编码规范文件：分析编码规范，确保编码符合规范
